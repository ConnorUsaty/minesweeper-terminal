# Minesweeper-Terminal
A simple minesweeper game that can be played in the terminal.

<h3>Features:</h3>
<ul>
 <li>Custom board sizing</li>
 <li>Difficulty selection</li>
 <li>Uses DFS to expand the safe area of "zero-count" squares</li>
</ul>
